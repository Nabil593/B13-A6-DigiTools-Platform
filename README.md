# DigiTools Platform

A premium, high-performance digital marketplace built with a minimalist aesthetic. This platform serves as a curated collection of professional digital assets, featuring a seamless user experience and pixel-perfect layouts.

---

## 📖 Description

**DigiTools Platform** is a modern web application designed for creators and developers. It provides a sophisticated interface to browse, filter, and manage premium digital products. Inspired by the clean and high-end design patterns of **Vercel** and **Linear**, this project focuses on speed, accessibility, and a "masterpiece" UI/UX.

---

## 🛠️ Technology Stack

This project is built using a modern frontend stack to ensure a fast, responsive, and interactive user experience:

* **Core Library:** [React.js](https://reactjs.org/) (Functional Components, state & props)
* **Build Tool:** [Vite](https://vitejs.dev/) (For ultra-fast development and bundling)
* **Styling & UI:** * [Tailwind CSS](https://tailwindcss.com/) (Utility-first CSS framework)
    * [DaisyUI](https://daisyui.com/) (Component library for Tailwind)
* **Language:** [JavaScript (ES6+)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
* **Notifications:** [React-Toastify](https://fkhadra.github.io/react-toastify/introduction) (For elegant user feedback and alerts)
* **Data Management:** [JSON](https://www.json.org/json-en.html) (Structured local database for product information)

---

## ✨ Key Features

### 1. Tab-Based Navigation System
- **Centralized Controls:** Two buttons at the center of the UI to toggle between **Products** and **Cart** sections.
- **Dynamic Visibility:** The **Product** section is visible by default, providing an immediate browsing experience for the user.
- **Empty State Handling:** The **Cart** section displays a clean "Empty Message" by default to guide users toward adding products.

### 2. Advanced Cart Management
- **Navbar Integration:** Real-time cart count updates in the Navbar as users add items. Clicking the Navbar cart icon instantly directs users to the `cart-section`.
- **Interactive Feedback:** Clicking the **Buy Now** button provides instant visual feedback by changing the button text to "Added to Cart".
- **Dynamic Totals:** Live calculation of the total price for all selected products within the cart.
- **Removal & Reset:** Users can remove individual products from the cart or use the **Proceed to Checkout** button to clear the entire cart and reset the Navbar count.

### 3. UI/UX Design
- Built with a focus on minimalism, featuring a responsive grid layout and smooth user interactions powered by Tailwind CSS, DaisyUI and React state management.
